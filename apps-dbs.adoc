== Oracle
// 
*link:https://www.netapp.com/pdf.html?item=/media/8744-tr3633pdf.pdf[TR-3633: Oracle databases on ONTAP]* - This document addresses the requirements for operating an Oracle database on NetApp storage in two ways. This document first discusses general considerations for all environments followed by specific recommendations based on the choice of virtualization and OS.
