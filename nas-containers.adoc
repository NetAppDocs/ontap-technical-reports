== FlexCache
// Aug 2022 - 9.11.1 - Elliott Ecton
*link:https://www.netapp.com/pdf.html?item=/media/7336-tr4743.pdf[TR-4743: FlexCache in ONTAP^]* - NetApp FlexCache is a caching technology that creates sparse, writable replicas of volumes on the same or different NetApp ONTAP clusters. It can bring data and files closer to the user for faster throughput with a smaller footprint. Learn about how FlexCache technology works and provides best practices, limits, recommendations, and considerations for design and implementation.
